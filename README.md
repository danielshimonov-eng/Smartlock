# ESP32 Smart Lock

A Smart Lock system using:

- ESP32
- Servo Motor
- Keypad
- Website Control

Features:
- Password unlocking
- Mobile phone control
- Automatic locking
- No physical keys needed

## Components
- ESP32
- Servo Motor
- 4x4 Keypad
- Jumper Wires
- Power Supply

## How It Works
The ESP32 hosts a website that allows users to lock and unlock the door from a phone.
